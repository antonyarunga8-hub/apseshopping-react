# ApseShopping React

A React-based multi-vendor e-commerce web app built with Vite, 
featuring product listings, retail, wholesale, and services pages.

## Live Demo
[apseshopping-react.vercel.app](https://apseshopping-react.vercel.app)

## Tech Stack
- React + Vite
- React Router (protected routes)
- Context API (Cart, Wishlist, Toast)
- Web Crypto API (PBKDF2 auth)
- Deployed on Vercel

## Features
- Multi-vendor product listings
- Retail, wholesale & services pages
- Route protection & admin panel
- Wishlist & cart functionality
- Responsive UI/UX

## Setup
```bash
npm install
npm run dev
```
